UV Exposure API Documentation

Leverage our powerful UV Exposure API to do amazing things!

With our UV Exposure API you can request average UV exposure by two different methods (ZIP code or Latitude and Longitude)!

How do I get access?
  • Request for an API key, by clicking buttom below.
  • Use the API key provided to get UV information you need.
  • You can also Sign Up for an account, if you want to upload your own UV exposure information

Base URL

http://ec2-18-221-206-206.us-east-2.compute.amazonaws.com:3000/api/averageUV
https://ec2-18-221-206-206.us-east-2.compute.amazonaws.com:8080/api/averageUV
Supported Methods: GET

Request Parameters

key=[key] (REQUIRED)
  • key - Your API Key.

API Endpoints

Description Required Parameters Example(s)
Get observation by lat/lon lat,lon &lat=38.123&lon=-78.543
Get observation by zip code zipcode &zipcode=85718

Example Request:

https://ec2-18-221-206-206.us-east-2.compute.amazonaws.com:8080/api/averageUV?zipcode=85718&key={API_KEY}
